# PoE2 ValueFilter — live price reference

A mobile-friendly quick reference of Path of Exile 2 item values for the current league,
served via GitHub Pages:

**→ https://thaluffy.github.io/poe2-prices/**

- **Headline** = the live [poe2scout](https://poe2scout.com) spot price.
- **Under the name** = the HLR-WM filter value (a robust, freshness- & liquidity-weighted
  estimate) and its Δ vs the live spot.
- Search, sort, and tap the tier chips to filter. Works offline once loaded.

This repo contains only the rendered `index.html` report — a snapshot regenerated on a
schedule from public market data. No game files or filter source live here.
